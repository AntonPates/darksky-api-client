## Build

``` 
dep ensure
go build -o darksky-api-client
```

## Run

```
darksky-api-client -config path_to_config.yaml
```

## Locations

Locations are enumerated in config (see config.yaml)

## API key

You can get it on [darksky.net](https://darksky.net)
